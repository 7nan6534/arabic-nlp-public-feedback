# arabic-nlp-public-feedback# Arabic NLP for Public Feedback Analysis

## Overview
This project analyses Arabic public feedback data to extract sentiment and identify key issues raised by citizens.
The goal is to support decision-makers in improving public services.

## Objectives
- Clean and preprocess Arabic text
- Analyse sentiment distribution
- Compare complaints across service categories
- Provide actionable insights for policy improvement

## Tools
- Python
- Pandas
- Regex
- Matplotlib

## Key Insights
- Health and education receive the highest negative feedback.
- Digital services show improvement but still attract complaints.
